# Quick Start

Install cookiecutter with [pipx](https://pipxproject.github.io/pipx/installation/)


```bash
$ pipx install cookiecutter
```

Generate the boilerplate cookiecutter:

```bash
$ cookiecutter https://github.com/mrcartoonster/cookie_flask.git
```
